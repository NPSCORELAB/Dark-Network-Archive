Introduction

Two networks at different times, 2006 and 2015, of FIFA Standing Committee membership. This dataset was procurred from UCINET’s site and is described as ‘overt networks with covert elements’. The initial data was reconstructed by Gemma Edwards using FIFA’s Committee 2006 activity report. This data consists of 710 nodes in an directed network that represents joint memebrship of a standing committee.

Abstract

These data were reconstructed by Gemma Edwards using FIFA’s Committee 2006 activity report. Each actor was tied to a standing committee in a bimodal network. The data procurred from UCINET’s site (https://sites.google.com/site/ucinetsoftware/datasets/covert-networks/fifa) stored these networks as one-mode graphs in which actors (people) are inteconnected via ‘co-membership’ ties.
